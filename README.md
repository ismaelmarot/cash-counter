##🚀 Getting Started

### Prerequisites

- Node.js >= 18.12
- npm or yarn

### Installation
```bash
git clone https://github.com/ismaelmarot/cash-counter.git
cd cash-counter
npm install
```

## Running the App
```bash
npm run dev
```
Open your browser at http://localhost:5173
